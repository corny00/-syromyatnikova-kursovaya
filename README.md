Курсовая работа на тему: «Автоматизация учёта и выдачи задержанных транспортных средств на специализированной стоянке»


---Описание проекта
Программа предназначена для автоматизации работы специализированной стоянки ГИБДД. Система позволяет регистрировать поступление эвакуированных автомобилей, фиксировать повреждения, рассчитывать стоимость хранения, формировать акты приёма-передачи и вести учёт платежей.

Решает следующие задачи:
1.Учёт всех поступивших на стоянку транспортных средств
2.Фото состояния автомобиля при поступлении
3.Автоматический расчёт стоимости хранения с учётом тарифов
4.Формирование официальных актов при выдаче автомобиля владельцу
5.Разграничение прав доступа между инспектором ГИБДД, диспетчером стоянки и администратором системы


---Технологии
Язык программирования: C#
Платформа: WPF (.NET Framework / .NET 6+)
СУБД: Microsoft SQL Server (SSMS)
Доступ к данным: ADO.NET (SqlConnection, SqlCommand, SqlDataAdapter)
Генерация документов: Xceed DocX (или HTML-экспорт в .docx)
Инструменты разработки: Visual Studio 2022, Git


---Как развернуть и запустить проект
1. Установить необходимое ПО
Установить Visual Studio 2022 Community с нагрузкой «.NET desktop development»
Установить SQL Server Express и SQL Server Management Studio (SSMS)
2. Развернуть базу данных
Открыть SQL Server Management Studio → подключиться к серверу (обычно (local)\SQLEXPRESS или .)
Нажать кнопку «New Query»
Открыть файл database.sql из папки проекта
Выполнить скрипт кнопкой «Execute» (зелёный треугольник)
При необходимости выполнить скрипт seed_data.sql для наполнения справочников
3. Настроить подключение к базе данных
Открыть проект в Visual Studio (файл *.sln)
В файле App.config проверить строку подключения:
Server=(local);Database=GBDD_BD;Trusted_Connection=True;
Изменить (local) на имя вашего экземпляра SQL Server
4. Запустить проект
Нажать F5 или кнопку «Пуск» в Visual Studio
При первом запуске система предложит авторизоваться
5. Учётные данные для входа (тестовые)
Администратор: логин admin, пароль admin123
Диспетчер: логин dispatcher, пароль disp123
Инспектор: логин inspector, пароль insp123


---Работа с программой
1.Регистрация нового автомобиля
На главной странице нажать кнопку «Принять новый автомобиль»
Заполнить обязательные поля: госномер, марка, модель, категория ТС
При необходимости указать владельца, инспектора, эвакуатор
Отметить зоны с повреждениями в таблице «Осмотр повреждений»
Нажать «Принять на стоянку» для сохранения записи
2.Просмотр и редактирование записей
В реестре на главной странице выбрать автомобиль двойным кликом
В карточке автомобиля отображаются: фото, время на стоянке, расчёт стоимости, данные владельца
Кнопка «Выдать автомобиль» доступна только диспетчеру и администратору
3.Фотофиксация
В карточке автомобиля нажать «Добавить фото» для загрузки снимков состояния ТС
Фото хранятся в папке Photos/ в директории приложения
Клик по фото открывает окно просмотра с возможностью масштабирования
4.Формирование акта при выдаче
На странице выдачи отметить чек-боксы проверки документов
Выбрать способ оплаты
Нажать «Сформировать квитанцию и выдать»
Акт в формате .docx автоматически сохранится на рабочем столе и откроется в Word
5.Отчёты (доступны администратору и диспетчеру)
Перейти в раздел «Отчёты» через верхнее меню
Выбрать период формирования
Просмотреть финансовые показатели, статистику по инспекторам и маркам авто
Экспортировать детализацию в файл .csv
6.Администрирование (только администратор)
Раздел «Администрирование» позволяет управлять пользователями системы
Добавление/редактирование сотрудников, назначение ролей, сброс паролей
Настройка тарифов на эвакуацию и хранение по категориям ТС
